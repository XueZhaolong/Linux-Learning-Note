# 4.8 Linux常用命令-关机重启命令
### 名称：shutdown
- 选项：
-c：取消前一个关机命令
-h：关机
-r：重启

### 其他关机命令
- halt
- poweroff
- init 0

### 其他重启命令
- reboot
- init 6

### 系统运行级别
- 0：关机
- 1：单用户（类似于Windows中F8安全模式）
- 2：不完全多用户，不含NFS服务（网络文件系统，文件共享）
- 3：完全多用户
- 4：未分配
- 5：图形界面
- 6：重启
- cat /etc/initttab：修改系统默认运行级别
- runlevel：查询系统运行级别

### 退出登录命令：logout
