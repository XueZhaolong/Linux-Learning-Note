# 4.3 Linux常用命令——文件搜索命令
## 4.3.1 文件搜索命令find
### 名称：find
**所在路径：** /bin/find

**执行权限：** 所有用户

**语法：** find [搜索范围] [匹配条件]

**功能描述：** 文件搜索

**范例：** 
- **根据名称查找：**
find /etc -name init

==在目录/etc中查找文件init==

find /etc -name * init *

==在目录/etc中查找名中包含init的文件==

find /etc -name init*

==在目录/etc中查找以init开头的文件==

find /etc -name init???

==在目录/etc中查找以init开头，后面有三个字符的文件==

find /etc -iname init*

==在目录/etc中查找以init开头的文件，不区分大小写==

find /etc -name inittab -exec ls -l {} \

==在/etc下查找inittab文件并显示其详细信息==

==-exec/-ok 命令 {} \：对搜索结果执行操作==

- **根据大小查找：**
1. find / -size +204800

==在根目录下查找大于100MB的文件==

==在Linux系统中，一个数据块为512字节，即0.5KB==

**+n 大于， -n 小于， n 等于**

2. find /etc -size +163840 -a -size -204800

==在/etc下查找大于80MB小于100MB的文件==

**-a 两个条件同时满足**

**-o 两个条件满足任意一个即可**

- **根据所有者查找：**

find /home -user shenchao

==在根目录下查找所有者为shenchao的文件，-group 根据所属组查找==

- **根据时间查找：**

find /etc -cmin -5

==在/etc下查找5分钟内被修改过属性的文件和目录==

-amin 访问时间access

-cmin 文件属性change

-mmin 文件内容modify

- **-type 根据文件类型查找**

==f 文件， d 目录， l 软连接文件==

- **-inum 根据i节点查找**

## 4.3.2 其他文件搜索命令
### 名称：locate
**命令所在路径：** /usr/bin/locate

**执行权限：** 所有用户

**语法：** locate 文件名

**功能描述：** 在文件资料库中查找文件

**范例：** locate inittab

**注意点！！！**
- 新创建的文件locate是找不到的，因为资料库中没有该文件，需要调用命令**updatedb**实时更新
- locate查找文件时严格区分大小写，如果不区分的话在locate后加上 -i
- 创建在/tmp目录下的文件即使updatedb后也找不到，因为/tmp临时目录中的文件不在文件资料库的收录范围内

### 名称：which
**命令所在路径：** /usr/bin/which

**执行权限：** 所有用户

**语法：** which 命令

**功能描述：** 搜索命令所在目录及别名信息

**范例：** which ls


### 名称：whereis
**命令所在路径：** /usr/bin/whereis

**执行权限：** 所有用户

**语法：** whereis [命令名称]

**功能描述：** 搜索命令所在目录及帮助文档路径

**范例：** whereis ls

### 名称：grep
**命令所在路径：** /bin/grep

**执行权限：** 所有用户

**语法：** grep -iv [指定字串] [文件]

**功能描述：** 在文件中搜寻字串匹配的行并输出

-i 不区分大小写

-v 排除指定字串

**范例：** grep mysql /root/install.log

==如果要排除以某个负号开头的行，需要改写为grep i- ^#，在需要排除的符号前加上三角（数字6）==

**在Linux中两种帮助的类型：1 命令的帮助； 5 配置文件的帮助**

**系统默认先看命令的帮助，如果想看配置文件的帮助，使用man 5 命令**
